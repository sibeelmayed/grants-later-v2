# Contributing

## Development
Prereqs: Node 20+ recommended.

```bash
npm ci
npm run dev
```

## Scripts
```bash
npm run lint
npm run build
```

## Branches
- Use `feat/*`, `fix/*`, `docs/*`, `chore/*`.

## Commits
Use Conventional Commits (e.g., `feat: ...`, `fix: ...`, `chore: ...`).

## Pull requests
- Keep PRs small and focused.
- Ensure CI is green before requesting review.
- Do not include secrets.
